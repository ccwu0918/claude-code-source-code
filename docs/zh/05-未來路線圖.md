## 2. KAIROS — 自主代理模式

最大的未發布功能，KAIROS 將 Claude Code 從被動助手轉變為主動自主代理。

### System Prompt（節選）

```
// src/constants/prompts.ts:860-913

你正在自主執行。
你會收到 <tick> 提示讓你保持活躍。
如果沒有用的事可做，呼叫 SleepTool。
傾向行動 — 讀取檔案、做變更、提交，無需詢問。

## 終端機焦點
- 未聚焦: 使用者離開了。大幅傾向自主行動。
- 聚焦: 使用者在看。更協作。
```

### 關聯工具

| 工具 | Feature Flag | 用途 |
|------|-------------|------|
| SleepTool | KAIROS / PROACTIVE | 控制自主操作間的節奏 |
| SendUserFileTool | KAIROS | 主動向使用者傳送檔案 |
| PushNotificationTool | KAIROS / KAIROS_PUSH_NOTIFICATION | 推播通知到使用者裝置 |
| SubscribePRTool | KAIROS_GITHUB_WEBHOOKS | 訂閱 GitHub PR webhook 事件 |
| BriefTool | KAIROS_BRIEF | 主動狀態更新 |

### 行為特徵

- 透過 `<tick>` 心跳提示保持活躍
- 根據終端機焦點狀態調整自主程度
- 可以獨立 commit、push 和做決策
- 傳送主動通知和狀態更新
- 監控 GitHub PR 變更

## 3. 語音模式

Push-to-talk 語音輸入已完全實現，但透過 `VOICE_MODE` feature flag 閘控。

```
// src/voice/voiceModeEnabled.ts
// 連接 Anthropic 的 voice_stream WebSocket 端點
// 使用 conversation_engine 模型做語音轉文字
// 按住快速鍵錄音，放開提交
```

- 僅限 OAuth 使用者（不支援 API Key / Bedrock / Vertex）
- 使用 mTLS WebSocket 連接

## 6. Buddy 系統（虛擬寵物）

完整的寵物夥伴系統已實現但尚未上線：

- **18 個物種**: 鴨子、鵝、blob、貓、龍、章魚、貓頭鷹、企鵝、烏龜、蝸牛、幽靈、墨西哥鈍口螈、水豚、仙人掌、機器人、兔子、蘑菇、chonk
- **5 檔稀有度**: 普通 (60%)、非凡 (25%)、稀有 (10%)、史詩 (4%)、傳說 (1%)
- **7 種帽子**: 皇冠、禮帽、螺旋帽、光環、巫師帽、毛線帽、小鴨子帽
- **5 項屬性**: DEBUGGING、PATIENCE、CHAOS、WISDOM、SNARK
- **1% 閃亮機率**: 任何物種的閃光變種
- **決定性生成**: 基於使用者 ID 雜湊

來源: `src/buddy/`

## 總結：三大方向

1. **新模型**: Numbat（下一代）、Opus 4.7、Sonnet 4.8 開發中
2. **自主代理**: KAIROS 模式 — 無人值守執行、主動行動、推播通知
3. **多模態**: 語音輸入就緒、瀏覽器工具待上線、工作流程自動化即將到來

Claude Code 正在從一個**程式設計助手**進化為一個**全天候自主開發代理**。
